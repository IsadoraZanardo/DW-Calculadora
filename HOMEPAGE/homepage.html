Esse projeto foi desenvolvido como atividade extracurricular individual do primeiro semestre da matéria de Desenvolvimento Web.

As ferramentas utilizadas foram HTML, CSS e JavaScript. Também vale-se dizer que recursos como HTML Tutorial e Bootstrap foram implementados em grande quantidade para, além de facilitar e otimizar o processo, possibilitar a aquisição de novos conhecimentos relacionados à disciplina.

A nomenclatura CS$aver foi inspirada em um trocadilho entre a junção das palavras "CSS" (Cascading Style Sheets) e "saver" (sinônimo de "guardar dinheiro" em inglês).

O objetivo da calculadora de finanças é uma aplicação web onde o usuário pode inserir diferentes informações sobre suas despesas, entre elas: valor de uma compra, categoria da forma de pagamento, data de aquisição e categoria do produto. Depois do user preencher obrigatoriamente todos esses campos mencionados, o sistema gera uma tabela dinâmica com todas as despesas lançadas e implementa uma função de soma do total gasto até o momento.

Ademais, a aplicação é composta por uma navbar, onde é possível trafegar entre Homepage (essa página), Add (calculadora) e Months (visualizar cálculo de meses anteriores dentro de um único ano).

Para finalizar, vale-se dizer que todos os dados e informações utilizados para preencher as tabelas são puramente fictícios e considera-se a data de hoje (12 de junho de 2025) como última atualização feita pelo cliente.
